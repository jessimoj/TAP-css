How to get started

Link your TAP-files to your projekt by making a link-tag in the html-documents head. For example, if you’ve put the files you want to use in a folder named ”css”, the link-tag should look like this:
<link href="css/tidy.css" rel="stylesheet" type="text/css">
or, if you want to use the playful theme, you use this one:
<link href="css/playful.css" rel="stylesheet" type=”text/css”>
 Note that you’ll always need to have the flexboxes.css, basics.css and colorthemes.css files in your css folder, but you only need the tidy or playful stylesheet, depending on which one you use - if you don’t want to use both, that is. Which we would totally understand. If you want to, you can even go ahead and skip both of them as well, and just rely on the basic stylesheet and the flexbox layouts. THE POSSIBILITIES ARE ALMOST ENDLESS.
(Well, no. Not really. This is not that big of a framework, it’s more like a quick and nice way to get started with a project. But still.)

Flexbox layout

To start with, you need a container. The allmighty flexbox container. Without it, you’re lost, pal. All of your flex elements needs to go into this one. A good place to put it would be directly under your documents <body>-tag. <div class=”container”> </div> For a simple and clean centered single column, use the .main-single class together with the .single class. <div class=”container”> <div class=”main-single single”> YOUR CONTENT HERE </div> </div> Want a multi column layout? Sure thing, buddy!
<div class="container row-contain"> <div class="row-col-main col-b ”>
This div will be the main column (because of the .row-col-main class) and will be placed as the second column (since it’s also got the .col-b class). </div> </div>

Use the col-# classes on your divs to change the order of them! They are as follows:
.col-a = order #1
.col-b = order #2
.col-c = order #3
.col-d = order #4

Stylesheet elements ("global")

Except from the usual tags and elements, the TAP-stylesheets do have some stuff you may find useful. These elements will (as you may already have figured out) look a bit different depending on which of the stylesheets you decide to use. To make it easier for you to switch between the two stylesheets, these elements don’t have different class names depending on which ”look” you would want - it’s all depending on which stylesheet you are currently using: hence the ”global” heading.

The box

Sometimes you want something to be a bit more eye-catching, whether it’s some extra important information or just something that deserves a fabulous box to hang out inside of. There are no rules for the box! Well, except the ones your stylesheet sets, of course.
Just give your div a .box-class and you got it!
<div class=”box”> </div>
Tip: We really like to use this class together with our forms (.form).

Media

Got media that you want centered? Use the .media-center class!
Like this example:
<img class="media-center" src=”pics/cat.gif” alt="compute cat" /> </span>

Sources

Use the .source-class to get a right-aligned, italic styled citation under for example a blockquote o a <p>-tag.
<span class="source">Wikipedia </span>

The navbar

To make your navbar appear all flexy and nice, use the .nav and .main-nav classes.

</ul>

Super big text

The bigger the better, right? Well, sometimes you really just want to get your message through and why not do that by using really big letters? Try the .super-big class! It will be fun!

Centered text

Use the class .centered and you got it!

The story class

Adding this class (.story) to a div will make the first letter in every <p>-tag look a little bit more ”dramatic”, like in a storybook. This is a good one to make longer texts feel a bit more interesting.

The tidy stylesheet

This stylesheet is supposed to give you a more clean yet sleek look on your webpage. Nothing extraordinary, but not just a simple view that gets your visitors bored.

The playful stylesheet

This stylesheet is made for a bit more quirky and fun look and feel.

The color themes

By applying class names to your div’s you can easily dress up your site with some nice color features. Nothing super fancy-schmancy or super dramatic, just a bit more... let’s say COLORFUL. Because sometimes color makes stuff a bit more fun, that’s just all. For the moment there’s just a few themes, but if you stick around there may be more comin’.

Note that you can mix the color themes by adding different classes to your divs! Like, why not a pink themed column next to a light gray one? Fabulous.
For now, if you feel like spicin’ it up a bit from the allready very pleasant basic look, you can apply the following color themes as classes on your divs:

The "Be Light" Theme

Gray shades to give you a crispy, smooth look.
Use class: .be-light
To style li-elements in the navbar, use class: .be-light-link

The "Shades of pink" Theme

Pink is nice. This is, as the name kind of foretells, a theme with different shades of pink mixed with white details.
Use class: .shades-of-pink
To style li-elements in the navbar, use class: .shades-of-pink-link
To get a matching theme on your box-element, use class: .shades-of-pink-box

The "Aqua" Theme

Cool shades of aquamarine.
Use class: .aqua
To style li-elements in the navbar, use class: .aqua-link
