Documentation (or: How to tap)

How to get started

Link your TAP-files to your projekt by making a link-tag in the html-documents head. For example, if you’ve put the files you want to use in a folder named ”css”, the link-tag should look like this: 

<link href="css/tidy.css" rel="stylesheet" type="text/css">

or, if you want to use the playful theme, you use this one: 
 <link href="css/playful.css" rel="stylesheet" type=”text/css”>

 Note that you’ll always need to have the flexboxes.css, basics.css and colorthemes.css files in your css folder, but you only need the tidy or playful stylesheet, depending on which one you use - if you don’t want to use both, that is. Which we would totally understand. If you want to, you can even go ahead and skip both of them as well, an just rely on the basic stylesheet and the flexbox layouts. THE POSSIBILITIES ARE ALMOST ENDLESS. (Well, no. Not really. This is not that big of a framework, it’s more like a quick and nice way to get started with a project. But still.)
